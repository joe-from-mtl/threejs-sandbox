Demo : Bvecs Viewer

This is an example of 15 bvecs orientation generated for a single shell with a weight=1. These points were generated using an electrostatic repulsion model to get a uniform angular coverage (see GitHub/scilpy for more details about the implementation. This is based on the following paper :
  • E. Caruyer et al., "Design of multishell sampling schemes with uniform coverage in diffusion MRI", Magn. Reson. Med. 69(6), 1534-1540 (2013), URL.
